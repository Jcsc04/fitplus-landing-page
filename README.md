# FitPlus — AI-Powered Fitness

![FitPlus Banner](https://via.placeholder.com/800x400/0A0A0A/00C8FF?text=FitPlus+AI+Fitness)

**The smartest AI fitness coach ever built.**

Modern, sleek, and high-converting landing page for **FitPlus** — an AI-powered fitness app featuring personalised workouts, nutrition plans, wearable integration, and Bio Age testing.

## ✨ Features

- AI that adapts to your body, goals, injuries, and Bio Age
- 28-day personalised nutrition plans
- Full wearable integration (Apple Watch, Garmin, WHOOP, etc.)
- Trainer dashboard mode
- Beautiful dark modern design with smooth animations
